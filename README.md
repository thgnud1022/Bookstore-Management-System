# Bookstore-Management-System
Hệ thống quản lý cửa hàng cho Nhà sách Cô Nữ
1. Project Overview | Tổng Quan Dự Án: This project is a bookstore management system developed to digitize and automate daily business operations of a small bookstore.
  The system helps manage inventory, sales transactions, customers, suppliers, and business reports efficiently.
  ---
  Dự án xây dựng hệ thống quản lý nhà sách nhằm số hóa và tự động hóa các hoạt động kinh doanh hằng ngày của cửa hàng.
  Hệ thống hỗ trợ quản lý kho, bán hàng, khách hàng, nhà cung cấp và báo cáo thống kê hiệu quả.
  
2. Problem Statement | Bài Toán Đặt Ra:

  Co Nu Bookstore currently manages many operations manually, which leads to:
  - Data inconsistency
  - Slow sales processing
  - Difficulty tracking inventory
  - Hard-to-manage customer and supplier information
  This system aims to solve these issues by providing a centralized digital solution.
  ---
  Nhà sách cô Nữ hiện vẫn quản lý thủ công nên gặp các vấn đề:
  - Dữ liệu dễ sai sót
  - Bán hàng chậm khi đông khách
  - Khó kiểm soát tồn kho
  - Thông tin khách hàng và nhà cung cấp rời rạc
  Hệ thống được xây dựng nhằm giải quyết các vấn đề trên bằng giải pháp quản lý tập trung.

3. Features | Chức Năng Chính:

  3.1 Inventory Management | Quản Lý Hàng Hóa
    
  3.2 Sales Management | Quản Lý Bán Hàng
    
  3.3 Customer & Supplier Management | Quản Lý Khách Hàng & Nhà Cung Cấp
    
  3.4 Reporting | Báo Cáo Thống Kê
   
4. Database Design | Thiết Kế Cơ Sở Dữ Liệu:
  - Normalized relational database
  - Use stored procedures to handle business logic
  - Apply constraints and triggers to ensure data integrity
  ---
  - Cơ sở dữ liệu quan hệ chuẩn hóa
  - Sử dụng stored procedure xử lý nghiệp vụ
  - Áp dụng ràng buộc và trigger để đảm bảo toàn vẹn dữ liệu
5. Tech Stack | Công Nghệ Sử Dụng:
  - WinForms C#
  - SQL Server
  - Figma
  - Git & GitHub
